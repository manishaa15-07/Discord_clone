# Discord 2.0 💬

A full-stack Discord-inspired real-time chat application built to provide seamless communication through servers, channels, messaging, and authentication.

---

## 🚀 Features

* 🔐 User Authentication
* 👥 Create & Join Servers
* 💬 Real-Time Messaging
* 📁 Channel-Based Communication
* 🟢 Online/Offline User Status
* 📷 Media & Image Sharing
* 🌙 Responsive Modern UI
* ⚡ Fast and Optimized Performance

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Socket.io Client

### Backend

* Node.js
* Express.js
* Socket.io

### Database

* MongoDB

### Authentication
* Nodemailer
* JWT Authentication
* bcrypt.js

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/discord-clone.git
cd discord-clone
```

### 2️⃣ Install Dependencies

#### Frontend

```bash
cd client
npm install
```

#### Backend

```bash
cd ../server
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside the server folder and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
CLIENT_URL=http://localhost:3000
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm start
```
